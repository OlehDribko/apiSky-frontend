Overview
ApiSky is a production‑ready React frontend application providing authentication and post management features.  
The project is designed with scalability, maintainability, and clean UI consistency in mind.
---
Features
• Authentication: register, login, logout with token persistence
• Protected Routing: secure access to posts only for authenticated users
• Posts CRUD: create, list, delete posts with timestamp display
• UI/UX: unified light theme, responsive layout, toast notifications for feedback
• Navigation: conditional header links based on auth state, logo redirects to posts when logged in
---
Tech Stack
• React 18 + Vite
• Redux Toolkit for state management
• React Router v6 for routing
• React Toastify for notifications
• CSS Modules for scoped styling
